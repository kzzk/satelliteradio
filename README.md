# satelliteradio
satelliteradio version


此1.3版的收音机是为安装在自行车上。要在自行车上接收必须要三轴平稳，垂直水平旋转以及盘面水平平衡，有待完善。目前人们对网络太过依赖，几乎放弃卫星接收；我认为有几个原因，人们意识不到网络其实并不可靠，特别是灾难面前，研究表明每逢地震，你总是打不通电话，原因是同时使用电话网络的人数超出预期。公众聚集的地方也往往通讯不畅；另网络有多重的限制，无论带宽还是网关；最主要的是没有一种民用的可以移动的卫星接收端的出现。如果8k视频出来的时候你还坚持使用网络？不如使用移动的卫星接收系统（它最终会在汽车上实现）当然，你不用扔掉手机，它依然可以作为接收终端，作为卫星电视的屏幕使用(目前已经有公司实现其功能）。
